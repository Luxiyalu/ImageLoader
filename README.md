EasyImage
====

Load image with:

1. image tag
2. canvas
2. background

### Example

```html
<div class="emg" data-src="images/arrow.png" data-type="" data-alt=""></div>
```

data-type could be:

0. auto (default, todo)
    load with canvas on default, change to img tag on mobile devices
1. img
2. cvs (todo)
3. bg (todo)
